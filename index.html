Welcome to the MSP430F5529LP GitHub Project Page.

This project provides an Operating Environment and Example code for the Texas Instruments MSP430F5529 Launchpad Development Board. This project will continue to be updated with new files to easily add accessories to your MSP430 Launchpad as well, such as LCD displays, Accelerometers, NeoPixel strings, and much more.
